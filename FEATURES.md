# Features Overview

## Dashboard
- **Real-time Metrics**: View total items, low stock count, locations, and printers
- **Inventory Trend Chart**: Line chart showing inventory and allocated items over time
- **Supply Mix Pie Chart**: Visual breakdown of supply types
- **Monthly Allocation Bar Chart**: Track allocation by month

## Inventory Management
- **Add Items**: Create new inventory items with SKU, quantity, and minimum stock levels
- **Search & Filter**: Find items by name or SKU
- **Status Tracking**: Automatic status updates (In Stock, Low Stock, Out of Stock)
- **Edit Items**: Update item details
- **Delete Items**: Remove items from inventory
- **Export to CSV**: Download inventory data for external use

## Location Management
- **View Locations**: See all warehouse locations
- **Location Details**: View items and printers per location
- **Add Locations**: Create new locations
- **Location Statistics**: Track inventory and printer counts

## Printer Management
- **Track Printers**: Monitor all printers across locations
- **Printer Status**: Track active, maintenance, and offline status
- **Supply Tracking**: See what supplies each printer uses
- **Add Printers**: Register new printers
- **Update Printer Info**: Modify printer details

## Reporting & Analytics
- **Stock Consumption Trend**: Line chart showing consumption over time
- **Allocation by Location**: Bar chart comparing allocated vs used inventory
- **Low Stock Alerts**: Identify items below minimum stock levels
- **Export Reports**: Download reports for analysis

## Authentication & Security
- **Role-based Access**: Admin and Manager roles
- **Secure Login**: JWT token-based authentication
- **Session Management**: Automatic logout on inactivity
- **Password Protection**: Secure credential storage

## User Interface
- **Dark Theme**: Professional dark mode design
- **Responsive Design**: Works on desktop, tablet, and mobile
- **Smooth Animations**: Fade-in, slide-in, and hover effects
- **Intuitive Navigation**: Easy-to-use sidebar menu
- **Real-time Updates**: Instant data refresh

## API Features
- **RESTful Design**: Standard HTTP methods
- **JSON Responses**: Easy to parse and integrate
- **Error Handling**: Comprehensive error messages
- **Interactive Documentation**: Swagger UI and ReDoc
- **CORS Support**: Cross-origin requests enabled

## Data Management
- **Automatic Status Updates**: Items automatically marked as low stock
- **Stock Movements**: Track inventory in/out movements
- **Audit Trail**: Record of all changes
- **Data Validation**: Pydantic validation on all inputs
- **Database Integrity**: Foreign key constraints and relationships

## Performance

- **Smooth Animations**: 60fps transitions
- **Responsive UI**: Instant feedback on user actions
- **Efficient API**: Minimal data transfer

## Customization
